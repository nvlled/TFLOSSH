# Applications solutions for LineageOS and Ubuntu Touch

[F-Droid](https://f-droid.org) is a community-maintained software repository. The main repository, hosted by the project, features only free libre software. Other repositories can be added to F-Droid. The [Aurora Store](https://f-droid.org/en/packages/com.aurora.store) can be installed via F-Droid in order to download applications from the Google Play Store with an anonymized, generated Google account (previously [Yalp Store](https://f-droid.org/en/packages/com.github.yeriomin.yalpstore) was an option).

Some solutions offered depend on Nextcloud, specifically synchronisation of files, contacts and calendars. Introduction, setup and usage documentation on Nextcloud can be found [here](https://github.com/wdbm/resources_Nextcloud/blob/master/documentation/documentation.md).
